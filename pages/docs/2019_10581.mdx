RootCause:
贵司要求使用systemui中recent功能
Solution:
如电话沟通，贵司目前使用SearchLauncherQuickStep，需要使用systemui中recent功能。
目前有两种方案：
方案一：
贵司使用Launcher3NoQsb，然后再做以下修改：
packages/apps/Launcher3/quickstep/AndroidManifest.xml
... ...
        <service
            android:name="com.android.quickstep.TouchInteractionService"
+            android:enabled="false"
            android:permission="android.permission.STATUS_BAR_SERVICE" >
            <intent-filter>
                <action android:name="android.intent.action.QUICKSTEP_SERVICE" />
            </intent-filter>
        </service>
方案二：
launcher切换为SearchLauncher：
vendor/sprd/generic/misc/launchercfg/LauncherPackages.mk
... ...
        ifeq ($(strip $(TARGET_BUILD_VERSION)),gms)
-            LAUNCHER_PACKAGE_NAME := SearchLauncherQuickStep
+            LAUNCHER_PACKAGE_NAME := SearchLauncher
        else
            LAUNCHER_PACKAGE_NAME := Launcher3QuickStep
        endif
如使用方案二，编译出不是SearchLauncher，请检查贵司功能是否配置其他launcher覆盖了SearchLauncher。
